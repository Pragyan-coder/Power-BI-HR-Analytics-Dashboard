# 📊 HR Analytics BI Interactive Dashboard

### ✅ Project Details:

- Client – Zelloitte
- Domain – Human Resource (HR)
  
### ✅ About the Dataset:

It consists of Employee attendance data for Q2, April to June’ 2025. The data includes daily attendance records for each employee. The table represents 3 months of attendance data. The daily attendance is recorded in columns labelled with the date and month. The values in these columns indicate the type of attendance, such as "P" for Present, "WFH" for Work from Home, "PL" for Paid Leave, "SL" for Sick Leave, "LWP" for Leave Without Pay, "WO" for Weekly Off, and other specific leave types.

### ✅ Project Brief:

Performed an in-depth analysis of Employee Attendance & Performance Metrics for Q2

### ✅ Data sources: 

Extracted data from a variety of Excel & CSV files, imported it to Power BI to perform further transformations.

### ✅ Process of Data Gathering: 

1.	Understanding the business requirements and framing the right business questions that will lead to better insight generation
2.	Collected the required data from the Organisation's Databases – from Excel records, CSV files, etc.
3.	Imported those CSV files to Power BI for further Transformation and analysis.
4.	Performed extraction, ‘Data Modelling’, transformation, removed anomalies, etc.
5.	Refined the datasets with cleaned & transformed data
